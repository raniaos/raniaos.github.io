SmartPlate

Android Application

  • Technologies used: AI, Deep Learning, YOLOv5, TensorFlow, Image Processing
  • Project for: Final Project in University (Tugas Akhir)
  • Programming Language: Kotlin (Front-end) & Python (Back-end)

Description

SmartPlate is an app to detect Indonesia license plate from image to text. SmartPlate uses YOLOv5 to detect the object (license plate) and TensorFlow to classify the characters in the license plate. SmartPlate also uses a lot of image processing steps, like grayscale, binarization, image moment, opening operation, and also uses Python libraries.


Struggles

  • Collecting the dataset for training the classification model. I had to take over hunderds pictures of license plates, then perform image processing for all the pictures, and then classify each processed images to the classes.
  • Learning how to make YOLOv5 model from scratch. I had to make the YOLOv5 model too from scratch by manually draw the bounding box in each pictures.
  • Training the classification model datas. In this project, I used 3 classification models which are VGG16, ResNet50, and MobileNetV2. I also had to change a few parameters in the models so I had to train the models a few times and it took minimum 3 hours to train 1 model.
